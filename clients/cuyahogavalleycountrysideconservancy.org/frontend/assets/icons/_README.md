# Icon Library Conventions

- Store SVG-only assets in this tree.
- Keep filenames lowercase; use `-` or `_` separators.
- Paths are referenced from HTML as relative paths under `assets/icons`.
- Do not include scripts or external references inside SVG files.
- Brand marks live under `logos/` when needed.

The set here is trimmed to files **referenced by site HTML**; add new SVGs only when a page needs them.
